# Contributing

Thanks for contributing! Please follow these guidelines.

## Commits

Follow [Conventional Commits](https://www.conventionalcommits.org/):

```
feat(scope): add new thing
fix(scope): correct broken behaviour
docs: update README
```

No `Co-Authored-By` or AI attribution lines in commit messages.

## Pull Requests

- One logical change per PR.
- Link the related issue in the PR body (`Closes #N`).
- All CI checks must pass before requesting review.

## Code Standards

Follow the org coding standards in `.github/copilot-instructions.md`.
